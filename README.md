## 简体中文版
这是一个版本号计算小工具。

### 使用方法
如果您的英语水平过硬，请直接按照程序的提示文字操作。

如果您对自己的英语水平不自信，请根据以下步骤操作：

1. 输入您想要的版本号；只需要输入 Major 和 Minor 部分。
	> [!NOTE]
	> 您需要按照 `Major.Minor` 格式输入。 
2. 程序会根据当前系统时间，模仿 C# AssemblyVersion 规则，生成 Build 和 Revision 部分。
3. 程序将会输出一个版本号。
4. 如果看到“Press Enter to continue...”字样，按下 Enter 键就可以退出程序。
